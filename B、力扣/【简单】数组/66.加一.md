## 分析 

从数组最后一位向前遍历，如果+1对10取余不等于0，说明没有进位，可以直接出来了，出来再看第一位是不是0（是否涉及到最高位进位的问题）用新的vector去接1和后面的digits。
## 代码

```cpp
class Solution {

public:

    vector<int> plusOne(vector<int>& digits) {

        for(int i = digits.size()-1; i >= 0; i--){

            if((digits[i] = (digits[i]+1)%10) != 0){

                break;

            }

        }

        if(digits[0]==0){

            vector<int> res;

            res.push_back(1);

            res.insert(res.end(),digits.begin(),digits.end());

            return res;

        }

        return digits;

    }

};
```
