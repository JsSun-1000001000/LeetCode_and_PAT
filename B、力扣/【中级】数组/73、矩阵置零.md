## 分析

新建立一个m行 n列 数组，遍历一次matrix碰到0就对所在行列标记为1

再遍历一次，只要行列有0就置0

## 代码
```cpp
class Solution {

public:

    void setZeroes(vector<vector<int>>& matrix) {

        int m = matrix.size();

        int n = matrix[0].size();

  

        vector<int> row(m), col(n);

  

        for(int i = 0; i < m; i++){

            for(int j = 0; j < n; j++){

                if(!matrix[i][j]){

                    row[i] = col[j] = 1;

                }

            }

        }

        for(int i = 0; i < m; i++){

            for(int j = 0; j < n; j++){

                if(row[i]||col[j]){

                    matrix[i][j] = 0;

                }

            }

        }

    }

};
```
