## 分析

给二叉树，返回最大深度，先看是不是空的，不是的话，看左子树和右子树哪个深，返回深的+1。
## 代码
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        //空树
        if(root == nullptr)return 0;
        //非空
        //先计算左子树
        int LH = maxDepth(root->left);
        //计算右子树
        int RH = maxDepth(root->right);
        return 1+max(LH, RH);
    }
};
```
