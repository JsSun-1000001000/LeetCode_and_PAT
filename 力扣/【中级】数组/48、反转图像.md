## 分析

我是傻逼

傻逼是不会遍历二维数组的，尼玛

总而言之就是先斜对角线对置，再行上reverse
## 代码
```cpp
class Solution {

public:

    void rotate(vector<vector<int>>& matrix) {

        int n = matrix.size();

        for(int i=0;i<n;i++){

            for(int j=0;j<i;j++){

                swap(matrix[i][j],matrix[j][i]);

            }

        }

        for(int i=0;i<n;i++){

            reverse(matrix[i].begin(),matrix[i].end());

        }

    }

};
```
